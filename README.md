# Today_Learn
- 매일 공부하는 내용에 대한 업데이트 및 기록

### 목표
- 데이터사이언스 관련한 매일의 성장을 기록한다.
- 짜임새있는 공부를 할 수 있는 기록을 남긴다.

### 방식
- 매일 1회 이상 올린다.(오전 계획, 오후 실행내용)
- 그날의 Issue는 추후 다른 폴더에 올려 추가 공부를 진행한다.

### 폴더 구조
```
├── README.md
├── today_learn_xxxx : 매일 매일의 내용
├── issues : 그날의 공부내용 중 추가 issue사항 정리
└── week1 : 1주차 자료
    └── index.html
```
